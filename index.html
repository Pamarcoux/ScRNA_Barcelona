```{r}
VlnPlot(AllSample,
            features = c("LGALS9"),
            group.by = "Disease",
            pt.size = 0,
            raster = FALSE
)
```



**What Cell type are expressing GAL9? **
```{r}
FeaturePlot(AllSample,
            features = c("LGALS9"),
            reduction = "umap_all",
            split.by = "Disease",
            pt.size = 0.01,
            raster = FALSE
)
```

**GAL9 is expressed mostly by Monocytes, and to a lesser extent by CD4 T-Cells and NK Cells.**
